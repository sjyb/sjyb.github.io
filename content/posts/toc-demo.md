# 目录与图床功能演示

## 一、为什么要目录
长文没有目录很难读。

## 二、如何开启
PaperMod 在 config.yml 设 TableOfContents: true 即可。

### 2.1 全局开启
所有文章默认显示。

### 2.2 单篇开关
frontmatter 加 `toc: true/false` 覆盖。

## 三、图床怎么用
PicGo 上传后，Markdown 里直接粘贴返回的 GitHub 链接。

## 四、总结
目录 + 图床让博客更专业。
